# International Trip Packing List

*For international travel (1+ weeks)*

---

## Documents (CRITICAL)

- [ ] Passport (check expiration - needs 6 months validity)
- [ ] Visa (if required)
- [ ] Travel insurance docs
- [ ] Flight confirmations (printed + digital)
- [ ] Hotel confirmations
- [ ] Credit cards (notify bank of travel)
- [ ] Some local currency (or plan to get at arrival)
- [ ] Copies of important docs (separate from originals)

---

## Essentials

- [ ] Phone + charger
- [ ] Wallet / ID
- [ ] AirPods
- [ ] Medications (in original containers)
- [ ] Glasses / contacts + supplies

---

## Clothing

- [ ] Underwear (7+ days or plan to laundry)
- [ ] Socks (7+ days)
- [ ] T-shirts / casual tops
- [ ] Dress shirts (if needed)
- [ ] Pants / shorts
- [ ] Jacket / layers
- [ ] Sleepwear
- [ ] Shoes (2-3 pairs: casual, walking, dress)
- [ ] Belt

### For Weddings/Formal Events
- [ ] Suit / formal attire
- [ ] Dress shoes
- [ ] Tie / accessories
- [ ] Cufflinks

---

## Toiletries

- [ ] Toothbrush + toothpaste
- [ ] Deodorant
- [ ] Face wash
- [ ] Moisturizer
- [ ] Razor + supplies
- [ ] Hair product
- [ ] Sunscreen
- [ ] Any prescription items

*Note: Keep liquids in carry-on compliant sizes or check bag*

---

## Tech

- [ ] Phone charger
- [ ] Watch charger
- [ ] Laptop + charger (if needed)
- [ ] Camera (Fuji X100VI) + charger
- [ ] Extra SD cards
- [ ] **Power adapter** (Europe: Type C/F)
- [ ] Portable battery pack
- [ ] Headphones (wired backup for planes)

---

## Comfort / Misc

- [ ] Neck pillow (for long flights)
- [ ] Eye mask
- [ ] Ear plugs
- [ ] Book / Kindle
- [ ] Snacks for travel day
- [ ] Reusable water bottle (empty through security)
- [ ] Day bag / backpack

---

## Pre-Departure

- [ ] Notify credit card companies of travel
- [ ] Set up international phone plan
- [ ] Download offline maps
- [ ] Download translation app
- [ ] Exchange some currency (or know ATM plan)
- [ ] Water plants
- [ ] Take out trash
- [ ] Newton situation handled
- [ ] Mail hold (if long trip)
- [ ] Lock up
- [ ] Thermostat adjusted

---

## Carry-On Must-Haves

In case luggage is delayed:
- [ ] Change of underwear
- [ ] Change of shirt
- [ ] Toothbrush + toothpaste
- [ ] Phone charger
- [ ] Medications
- [ ] Valuables (camera, laptop)
- [ ] All documents
